# 🌊 NEON WAVE FESTIVAL 2026 - Landing Page

A premium, high-performance promotional website designed with a **"High-Voltage"** editorial aesthetic. This project merges ultra-vibrant neon colors with a brutalist layout to create an immersive digital experience.

## 🎨 Design Highlights
- **High-Voltage Palette:**
    - **Electric Cyan:** `#00F5FF` (Primary Glow)
    - **Neon Lime:** `#CCFF00` (Energy Accent)
    - **Hot Magenta:** `#FF00A0` (Vibrant Contrast)
    - **Deep Space:** `#020617` (Baseline Background)
- **Visuals:** Curated high-resolution artist imagery emphasizing a futuristic electronic and urban vibe.
- **Typography:** Using `Bebas Neue` for impactful headings and `Space Grotesk` for technical clarity.
- **Interactive Elements:** Glassmorphic cards, custom clip-path buttons, and smooth scroll-reveal animations.

```text
music-festival/
```

## 🚀 Technical Specifications
- **Modern CSS:** Implementation of CSS Variables, Grid, and Flexbox.
- **Performance:** Zero framework overhead (Pure Vanilla JS/CSS) for lightning-fast load times.
- **Animations:** `Intersection Observer API` manages the scroll-trigger animations to ensure a smooth, premium feel.
- **Responsiveness:** Mobile-first approach ensuring perfect rendering from 320px to 4K monitors.

## 🛠️ Quick Start
1. **Clone/Download** the repository.
2. **Open `index.html`** in any modern web browser.
3. **Note:** An active internet connection is required to fetch Google Fonts and Unsplash images.
