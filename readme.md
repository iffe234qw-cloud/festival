# 🌊 NEON WAVE FESTIVAL 2026 - Landing Page

A premium, high-performance promotional website designed with a **"Brutalist-meets-Luxury"** editorial aesthetic. This project demonstrates the power of modern CSS (Grid/Flexbox) and Vanilla JavaScript to create immersive, interactive experiences without the overhead of heavy frameworks.

## 🎨 Design Philosophy
The website is inspired by modern digital magazines and award-winning promotional sites. Key design pillars include:
- **Typography-First:** Using `Bebas Neue` for massive, impactful headlines and `Space Grotesk` for a futuristic, technical feel.
- **Asymmetry & Overlap:**Breaking the traditional "boxed" layout with overlapping imagery and shifted grids.
- **High-Voltage Palette:** A deep space `#020617` background contrasted with electrifying neon gradients: Electric Cyan (`#00F5FF`), Neon Lime (`#CCFF00`), and Hot Magenta (`#FF00A0`).
- **Glassmorphism:** Utilizing `backdrop-filter: blur()` to create depth and a sense of luxury in cards and navigation.

## 🚀 Technical Implementation

### Frontend Stack
- **HTML5:** Semantic tags (`<nav>`, `<section>`, `<footer>`) for SEO and Accessibility.
- **CSS3:** 
    - **Custom Properties:** Centralized color palette and transitions for easy branding updates.
    - **CSS Grid & Flexbox:** Complex overlapping layouts and responsive masonry grids.
    - **Clip-Path:** Used on buttons to create a unique, angled "futuristic" geometry.
- **Vanilla JavaScript:**
    - **Intersection Observer API:** Triggers "fade-up" reveal animations as the user scrolls.
    - **Dynamic DOM Manipulation:** Handles the festival schedule tab switching without page reloads.
    - **Event Listeners:** Manages the sticky navbar and mobile hamburger menu.

## 🛠️ Installation & Setup
Since this project uses only vanilla web technologies, no build step or installation is required.

1. **Clone or Download** the project folder.
2. **Open `index.html`** in any modern web browser (Chrome, Firefox, Edge, Safari).
3. **Assets:** All imagery is linked via high-resolution Unsplash URLs, so an internet connection is required to load the visuals.

## 📱 Responsive Design
The site is mobile-first and adaptive across:
- **Desktop (1440px+):** Full editorial layout with multi-column grids.
- **Tablet (768px - 1024px):** Simplified grids and adjusted typography sizing.
- **Mobile (< 768px):** Single-column stack, touch-optimized buttons, and a sliding hamburger menu.

## ♿ Accessibility (A11y)
- **Contrast:** High-contrast color ratios for readability.
- **Keyboard Nav:** Functional focus states for all interactive elements.
- **Semantics:** Proper heading hierarchy (`H1` $\rightarrow$ `H2` $\rightarrow$ `H3`) for screen readers.
